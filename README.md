# Malaria Detection using Deep Learning

## Description
This project detects malaria infected cells using deep learning CNN model.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

## Dataset
Malaria cell images dataset.

## How to Run
1. Install requirements
2. Run train.py
3. Use predict.py for detection
